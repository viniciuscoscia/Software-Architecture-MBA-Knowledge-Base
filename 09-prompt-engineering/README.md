## Técnicas a cobrir

- **Zero-Shot** — sem exemplos, só instrução
- **Few-Shot** — com exemplos no prompt
- **Chain of Thought (CoT)** — pedir raciocínio passo a passo
- **Tree of Thought (ToT)** — explorar múltiplos caminhos de raciocínio
- **ReAct** — Reasoning + Acting (agentes que agem com base no raciocínio)

## Aplicações práticas no desenvolvimento de software

- Geração de código
- Code review automatizado
- Criação de testes
- Documentação
- Debug assistido
